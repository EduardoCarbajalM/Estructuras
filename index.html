1. Pilas en Python

        class Pila:
            def __init__(self):
                self.items = []
        
            def esta_vacia(self):
                return len(self.items) == 0
        
            def apilar(self, item):
                self.items.append(item)
        
            def desapilar(self):
                if not self.esta_vacia():
                    return self.items.pop()
                else:
                    return None
        
        # Ejemplo de uso de una pila
        mi_pila = Pila()
        mi_pila.apilar(1)
        mi_pila.apilar(2)
        mi_pila.apilar(3)
        print(mi_pila.desapilar())  # Desapila el 3

Video

Descripción

Este código de Python define una clase llamada Pila. La clase tiene cuatro métodos:

__init__, esta_vacia, apilar y desapilar.

En el ejemplo de uso, se crea un objeto de la clase Pila llamado mi_pila. Luego se agregan los números 1, 2 y 3 a la pila utilizando el método apilar. Finalmente, se elimina y se imprime el último elemento de la pila utilizando el método desapilar.

2. Colas en Python

        from collections import deque

        mi_cola = deque()
        mi_cola.append(1)
        mi_cola.append(2)
        mi_cola.append(3)
        print(mi_cola.popleft())  # Desencola el 1

Video

Descripción

El código en Python utiliza la clase "deque" de la biblioteca "collections" para crear una cola FIFO.
Se agrega 1, 2 y 3 a la cola y se elimina e imprime el primer elemento con "popleft".

4. Arboles en Python

        class NodoArbol:
        def __init__(self, valor):
            self.valor = valor
            self.izquierda = None
            self.derecha = None
    
        # Ejemplo de un árbol binario
        raiz = NodoArbol(1)
        raiz.izquierda = NodoArbol(2)
        raiz.derecha = NodoArbol(3)

Video

Descripción

Este código crea una estructura de datos que simula un árbol binario. Cada "nodo" del árbol contiene un número o "valor". El nodo también puede tener dos ramas, una a la "izquierda" y otra a la "derecha," que a su vez son otros nodos.

En el ejemplo, creamos un árbol con tres nodos. El primer nodo es la "raíz" con un valor de 1. A continuación, le asignamos un nodo con el valor 2 a su izquierda y otro nodo con el valor 3 a su derecha. Esto representa una estructura de árbol sencilla con tres nodos interconectados.

5. Grafos en Python

        grafo = {
            'A': ['B', 'C'],
            'B': ['A', 'C', 'D'],
            'C': ['A', 'B', 'D'],
            'D': ['B', 'C']
        }
        
        # Obtener los vecinos del nodo 'B'
        print(grafo['B'])  # Devuelve ['A', 'C', 'D']

Video

Descripción

Este código representa un conjunto de conexiones entre nodos usando un grafo. Imagina que estos nodos son como estaciones en una red de transporte, y las conexiones son las rutas que conectan esas estaciones.

En este caso, el grafo tiene cuatro estaciones etiquetadas como 'A', 'B', 'C' y 'D'. Cada estación está conectada a otras estaciones, y esta información se almacena en el diccionario llamado "grafo."

Si quieres saber cuáles son las estaciones conectadas a la estación 'B', simplemente miras en el diccionario en la entrada 'B', y encontrarás una lista que contiene 'A', 'C' y 'D'. Esto significa que desde la estación 'B', puedes llegar directamente a las estaciones 'A', 'C' y 'D'.
